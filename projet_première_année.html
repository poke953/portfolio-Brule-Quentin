Projet de Première Année

Mon projet de première année consistait à réaliser un site web e-commerce pour une entreprise de vente en ligne d'ordinateurs.

Le projet a été réalisé en utilisant HTML, CSS et JavaScript pour le développement du site web, ainsi que PHP et MySQL pour la gestion de la base de données.
